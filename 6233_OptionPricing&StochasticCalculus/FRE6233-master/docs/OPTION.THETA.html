OPTION.THETA function

This article describes the formula syntax of the OPTION.THETA function.

Description

Return the option call (k > 0) or put (k < 0) theta.

Syntax

OPTION.THETA(v, S, sigma, option, k, t, r, _dt)

v is a handle to a variate.
S is the spot.
sigma is the volatility.
option is the contract type from OPTION_*.
k is the strike.
t is the time in years to expiration
r is the continuously compouned interest rate. Default is 0.
_dt is an optional time increment. Default is 1/250.

Option theta is the derivative of option value with respect to time.

Return to index.
