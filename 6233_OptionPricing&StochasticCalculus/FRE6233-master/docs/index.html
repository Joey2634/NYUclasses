Functions and macros of the FRE6233 add-in.

Documentation for FRE6233.

FRE6233 functions

OPTION.DELTA Return the option call or put delta.
OPTION.GAMMA Return the option call (k > 0) or put (k < 0) gamma.
OPTION.IMPLIED Return the option call (k > 0) or put (k < 0) implied vol.
OPTION.MONEYNESS Return the option moneyness.
OPTION.THETA Return the option call (k > 0) or put (k < 0) theta.
OPTION.VALUE Return the option value.
OPTION.VEGA Return the option call (k > 0) or put (k < 0) vega.

Variate functions

VARIATE.CDF Compute the Esscher transform of the cumulative distribution function.
VARIATE.CUMULANT Compute the cumulant of a variate.
\VARIATE.NORMAL Return a handle to a standard normal model.
