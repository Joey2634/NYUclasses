OPTION.DELTA function

This article describes the formula syntax of the OPTION.DELTA function.

Description

Return the option call or put delta.

Syntax

OPTION.DELTA(v, S, sigma, option, k, t, r)

v is a handle to a variate.
S is the spot.
sigma is the volatility.
option is the contract type from OPTION_*.
k is the strike.
t is the time in years to expiration.
r is the continuously compouned interest rate. Default is 0.

Option delta is the derivative of option value with respect to forward.

Return to index.
