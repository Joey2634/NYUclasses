XLL.REFTEXT function

This article describes the formula syntax of the XLL.REFTEXT function.

Description

Converts a reference to an absolute reference in the form of text.

Syntax

XLL.REFTEXT(ref, A1)

ref is a cell reference.
A1 is a boolean specifying A1 or R1C1-style references.

Use REFTEXT when you need to manipulate references with text functions. After manipulating the reference text, you can convert it back into a normal reference by using TEXTREF.

Return to index.
