VARIATE.CDF function

This article describes the formula syntax of the VARIATE.CDF function.

Description

Compute the Esscher transform of the cumulative distribution function.

Syntax

VARIATE.CDF(model, x, s, nx, ns)

model is a handle to a variate model.
x is the value at which to compute the CDF.
s is the Esscher parameter.
nx is the number of derivatives with respect to x. Default is 0.
ns is the number of derivatives with respect to s. Default is 0.

Return the Esscher transform of the cumulative distribution function and its derivatives.

Return to index.
