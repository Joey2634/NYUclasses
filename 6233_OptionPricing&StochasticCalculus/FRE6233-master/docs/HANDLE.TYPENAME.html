HANDLE.TYPENAME function

This article describes the formula syntax of the HANDLE.TYPENAME function.

Description

Return name of type corresponding to handle.

Syntax

HANDLE.TYPENAME(handle)

handle is a handle.

Return the C++ RTTI typeid(T).name() of a handle created with xll::handle<T>. This uses runtime type information to return the name of the derived class of T used to initialize the handle. If the handle is not found then #NUM! is returned.

Return to index.
